# 💰 Python Expense Tracker

A simple **Command-Line Expense Tracker** built using **Python** to help users record, organize, and analyze their daily expenses. This project focuses on strengthening Python fundamentals through file handling, CRUD operations, and data processing.

---

## 📌 Features

* ➕ Add New Expense
* 📋 View All Expenses
* ❌ Delete Expense
* 🔍 Search Expenses by Category
* 📊 Category-wise Expense Report
* 💰 Calculate Total Spending
* 📅 Monthly Expense Report
* 💾 Persistent Data Storage using File Handling
* ✅ Input Validation & Exception Handling

## 📖 How It Works

### ➕ Add Expense

Store a new expense with:

* Expense Name
* Amount
* Category
* Date

---

### 📋 View Expenses

Display all recorded expenses in a formatted table.

---

### ❌ Delete Expense

Delete an expense using its ID.

---

### 🔍 Search Expense

Search all expenses belonging to a specific category.

Example:

```text
Enter Category: Food
```

---

### 📊 Category-wise Expense Report

Example Output

```text
Food             750.00
Travel          450.00
Entertainment   300.00
```

---

### 💰 Total Spending

Example Output

```text
Total Expenses: 1500.00
```

---

### 📅 Monthly Report

Example Output

```text

```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Jazaa0208/Expense_Tracker.git
```

2. Navigate to the project folder

```bash
cd Expense_Tracker
```

3. Run the program

```bash
python Expense_Tracker.py
```
